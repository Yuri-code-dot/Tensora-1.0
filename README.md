Tensora-1.0
Free. Open-source. No subscriptions. Pure AI chat.
An elegant, open-source AI chat interface built with modern web technologies. Tensora-1.0 brings conversational AI to your fingertips without corporate paywalls or hidden costs.
✨ FEATURES 
Clean, responsive UI – Works seamlessly on desktop, tablet, and mobile
Open-source models – Powered by free, community-driven AI models
Fast & lightweight – No bloat, just pure conversation
Dark theme – Easy on the eyes, built for extended sessions
Real-time chat – Instant responses with streaming support
Zero tracking – Your conversations stay private
Self-hostable – Deploy it yourself, maintain full control

🔧 Tech Stack
Frontend: HTML5, CSS3, JavaScript (Vanilla)
Backend: Node.js, Express
AI Models: Open-source (Mistral, Llama, DeepSeek, etc.)
Deployment: GitHub Pages + Node.js server
Styling: Dark theme, responsive design
💬 How It Works
User sends a message → Chat input captured
Message sent to backend → Server processes request
AI inference → Open-source model generates response
Real-time streaming → Response appears in chat instantly
History maintained → Conversation context preserved
